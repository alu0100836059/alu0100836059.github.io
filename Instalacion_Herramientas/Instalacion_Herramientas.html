Instalacion de NodeJS en Windows

Node.js es un entorno de programacion en la capa del servidor basado en el lenguaje de programación Javascript, con I/O de datos en una arquitectura orientada a eventos y basado en el motor Javascript V8.
Para su instalacion:

Instalacion de ATOM

Atom, el editor de codigo desarrollado por la gente de Github.
La principal caracteristica de Atom es, segun sus creadores, que es totalmente “hackeable”, es decir, que se puede toquetear y modificar hasta el mismo core. Esto es posible gracias que, a pesar de ser una aplicacion de escritorio (lo que antiguamente se llamaba programa), esta realizada enteramente con tecnologias web. Ademas estara perfectamente integrado con Node.js lo que permitira el uso de los miles de paquetes y librerías que componen su repositorio ademas de los 50 paquetes que contendra por defecto su core.
Para su instalacion:

  • Dirijase a la página oficial de ATOM: https://atom.io/
  • Seleccione la descarga para windows.
  • Abra el ejecutable y siga las instrucciones del menu de instalación.
  • Ya puede ejecutar y disfrutar del editor de texto ATOM

Instalacion de Github Desktop para Windows

Para su instalacion:

  • Dirijase a la pagina oficial: https://desktop.github.com/
  • Seleccione la descarga para windows.
  • Abra el ejecutable y siga las instrucciones del menu de instalacion.
  • Ya puede ejecutar y disfrutar de la aplicacion de escritorio de Github

Utilizacion de Cloud9

C9 se trata de un programa que se ejecuta sobre el navegador pero que a pesar de ese limitante tiene las funcionalidades mas destacadas de un IDE en el que ademas tenemos un entorno de trabajo real donde podemos poner nuestros programas en ejecucion, ya sean sitios web o programas ejecutables.
Uno de los factores a destacar es la posibilidad de asociar nuestros repositorios de Github para poder trabajar sobre ellos desde Cloud9.
Para comenzar:

  • Dirijase a la pagina oficial: https://c9.io/
  • Si no posee una cuenta creela accediendo a Sing Up
  • Si ya posee una cuenta ingrese sus datos accediendo a Sing In
  • A continuacion ya puede asociar sus repositorios o empezar a trabajar en un nuevo proyecto.

Lenguaje Markdown

Markdown es un lenguaje de marcado que facilita la aplicacion de formato a un texto empleando una serie de caracteres de una forma especial. En principio, fue pensado para elaborar textos cuyo destino iba a ser la web con mas rapidez y sencillez que si estuviesemos empleando directamente HTML. Y si bien ese suele ser el mejor uso que podemos darle, tambien podemos emplearlo para cualquier tipo de texto, independientemente de cual vaya a ser su destino.
Existen multiplos recursos para la conversion directa y previsualizaciones del lenguaje Markdown en lo referente a html. En este caso yo me he servido de una herramienta web que destaca por su rapidez y claridad.
Stackedit.io/editor

  • Dirijase a la pagina oficial: https://stackedit.io/editor
  • Segun accedemos a la pagina podemos ver un archivo de ejemplo sobre el que empezar a trabajar.
  • Ademas, posee menus laterales asi como una barra superior donde se muestran los elementos mas utilizados y otras configuraciones.
  • Una vez acabado nuestro diseño en Markdown podremos tanto exportarlo como tal, convertir a html y guardarlo en nuestro pc o en la nube, etc.
